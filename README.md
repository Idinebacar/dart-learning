# Dart Learning Repository

👋 Hi, I'm **Chamsidine Bacar** — a beginner developer learning Flutter and Dart.  
This repository is my personal learning space where I practice Dart programming concepts step by step.

## 📂 Project Structure
- `basics/`  Variables, constants, data types
- `functions/`  Function syntax, recursion, optional & named parameters
- `conditions_and_loops/`  If-else, switch-case, for loops, greatest number
- `user_input/` Reading input, calculator, number input
- `oop/`  Classes, inheritance, polymorphism

## 🚀 How to Run
1. Install [Dart SDK](https://dart.dev/get-dart)  
2. (Optional) Install [Flutter SDK](https://docs.flutter.dev/get-started/install) — includes Dart SDK  
3. Download [Android Studio](https://developer.android.com/studio) for Flutter app development  
4. Navigate to the file folder  
5. Run with:
   ```bash
   dart filename.dart
